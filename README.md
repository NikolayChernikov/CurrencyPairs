# Currency pairs

## How to run locally

1. Clone repository `git clone git@github.com:NikolayChernikov/CurrencyPairs.git`
2. Goto folder `cd CurrencyPairs`
3. Run service `docker-compose up -d`

## Linters

1. Flake8 `flake8 bwg`
2. Isort `isort bwg`
3. PyLint `pylint bwg`
4. MyPy `mypy bwg`
